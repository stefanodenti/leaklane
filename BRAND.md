# LeakLane Brand Notes

## Positioning

LeakLane is a local-first security console for scanning GitHub and Bitbucket repositories with Gitleaks, preserving reports, and generating local AI triage through LM Studio.

## Brand Idea

Repositories enter a controlled lane before release or review. The mark combines two opposing lanes with a protected center point: source history comes in, findings are surfaced, and the operator decides the next action.

## Name

**LeakLane**

Short, direct, and connected to the existing pitlane workflow. It says what the tool does without sounding like a generic dashboard.

## Tagline

**Local secret triage for repository fleets.**

## Visual System

- Primary background: charcoal green, `#101613`
- Light surface: `#f6f7f4`
- Accent: security green, `#72b58e`
- Ink: `#202620`
- Radius: compact, 8px
- Typography: system sans for a native desktop feel

## Logo

The logo mark is an abstract lane switch:

- two crossing repository lanes
- one protected center point
- strong enough for app icon and favicon
- simple enough for small UI use
